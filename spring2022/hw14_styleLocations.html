Using Inline CSS

The benefits of using inline CSS include:

  • Allows for quick style changes to a single element in an HTML document.

Using CSS in the style Tag

The benefits of placing CSS between the style tags include:

  • Allows for more complex style changes to an entire HTML document or elements used more than once.
  • If an external stylesheet is used, CSS in the style tag can be used to override these values for only the document the changes are defined in.

Using an External Stuylesheet

The benefits of using an external stylesheet include:

  • Preventing your HTML document from becoming bloated with style attributes in individual elements or with the style tag by separating the presentation (CSS) from semantics (HTML).
  • Allowing the defined styles to be used over multiple HTML documents as long as they are linked to the stylesheet.
